# Brand logo

Primary file:

```
public/brand/logo.png
```

Also accepted (fallback):

```
public/logo.png
```

Used for the public site, admin, email masthead (inline CID), and favicon.

Prefer a **transparent PNG** (gold mark on clear background) so the logo sits cleanly on navy mastheads and dark UI. Avoid baked-in black squares.

Display sizes (approx):
- Public hero: ~160px
- Email masthead: 128px
- Welcome email: 112px
- Admin login / topbar: 120px / 64px
